## 推荐：恒源云 RTX 3090 ⭐

**理由：**

1. **每元产出最高**（排除存疑的 4090 价格）
2. **24GB 显存**远超需求（6-8GB），训练中不会 OOM
3. **¥1.30/h × 23h = ¥30**，跑 41M 步，接近完整的 50M 目标
4. **恒源云体验类似 AutoDL**：预装 PyTorch 镜像、秒级计费、SSH/JupyterLab
5. 如果 **缩减到 30M 步**：只需要 17h = **¥22**，还剩 ¥8 跑消融实验


## 恒源云操作流程

### 1. 注册
打开 [gpushare.com](https://gpushare.com) → 手机号注册

### 2. 充值
**先充 ¥30**（不要多充），支付宝/微信均可

### 3. 选实例
- GPU 市场 → 筛选 **RTX 3090**
- 计费方式：**按量计费**
- 镜像：选 PyTorch 2.x + Python 3.10+ 的预装镜像

### 4. 环境初始化（同 AutoDL 流程）
```bash
cd /root/gpushare-tmp  # 恒源云的数据盘路径
git clone <你的仓库> ATRI
cd ATRI
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install autorom -i https://pypi.tuna.tsinghua.edu.cn/simple
AutoROM --accept-license
```

### 5. 启动训练
```bash
tmux new -s dk
python train.py
# Ctrl+B D 断开
```

### 6. 关机
跑完立刻关机！恒源云按秒计费，关机即停。
